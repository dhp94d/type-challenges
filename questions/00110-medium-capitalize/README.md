**해설**
ts4.1에 도입된 템플릿 리터럴타입을 알고 있어야 풀 수 있습니다. 더불어 함꼐 나온 아래 유틸리티 타입또한 알고 있어야 합니다.
Uppercase<S>: 문자열 S의 모든 문자를 대문자로 변환합니다.
Lowercase<S>: 문자열 S의 모든 문자를 소문자로 변환합니다.
Capitalize<S>: 문자열 S의 첫 문자를 대문자로 변환합니다.
Uncapitalize<S>: 문자열 S의 첫 문자를 소문자로 변환합니다.
